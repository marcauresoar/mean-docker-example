Spreadsheet pattern for Jeep

AC_CODE;AC_DESC;AC_NAME;AC_PRICE;AC_INSTALLMENT_TYPE;AC_INSTALLMENT_PRICE;AC_VERSION;AC_ORDER;AC_TYPE

Attributes description:
  • AC_CODE: The unique code of the accessory. E.g. 50928103
  • AC_DESC: The description of the accessory. E.g. É o acessório certo para quem busca uma maneira exclusiva de se destacar dos demais, garantindo um design mais esportivo.
  • AC_NAME: The accessory's name. E.g. ENVELOPAMENTO DE TETO
  • AC_PRICE: The total price of the accessory (using dot instead of comma). E.g. 972.00
  • AC_INSTALLMENT_TYPE: The type of installment. E.g. 4x de
  • AC_INSTALLMENT_PRICE: The installment unity price. E.g. 243.00
  • AC_VERSION: The version of which the accessory belongs. E.g. A4N0
  • AC_ORDER: The order in which the accessory is going to appear in the list. E.g. 5
  • AC_TYPE: The assossiative type name of the accessory. E.g. EXTERIOR
